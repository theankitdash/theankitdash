Projects

Below are some of the projects I've worked on. If you'd like more details and gain hands-on experience about any of them, please feel free to contact me.

Personal AI Buddy Companion

AI-driven chatbot built with Gradio and Python to simulate a personalized digital companion. Designed for long-term engagement, with potential integration of deep learning and Langchain for enhanced personalization. It captures user preferences, stores contextual memory from conversations, and assists with daily tasks, reminders, and meaningful interactions.

AI Nutritional Health Assistant: Personalized Guidance for Indian Diets

AI-powered chatbot designed using Python to deliver personalized nutrition guidance based on user health metrics and dietary preferences, with a focus on Indian cuisine. Integrated Langchain and Gemma for context-aware conversations, and Redis for managing user data and chat history.

Personal Finance System for Expense Tracking and Analysis

JavaScript-based web application for tracking and categorizing daily expenses, offering detailed analysis to optimize personal financial planning. Built with Docker and Kubernetes for scalable, reliable performance in multi-user environments.

Administrator Controlled Timetable Based Automatic Facial Recognition Attendance System

Python-based facial recognition system for automated, subject-wise student attendance tracking using computer vision algorithms to enhance accuracy and efficiency.
